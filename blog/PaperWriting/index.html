Practice Insights about First Paper Writing

 2024-12-17 2024-12-17 blog 3 minutes read (About 489 words) 0 visits

前言

最近正在赶写一篇鲁棒性提示学习的文章，由于是个人的初次论文写作，从不知如何上手到亲身有所体会，我觉得还是有必要好好总结一下。

当然，个人也只是热衷于总结学习，第一次论文写作难免有不恰当、不准确的地方，如有错误，欢迎批评指正。

这篇博文写作比较随意，从一些感悟点入手，随后有什么就再次更新。

Introduction

我们先来将 Introduction 部分，从整体分布来看，一般分为以下几个部分：

  1. Background: 第一段主要是介绍所从事的研究领域以及现状如何，如果有新的研究领域提出就可以在这部分引入；
  2. Problem Introduction: 第二段主要是介绍所从事的研究领域中存在的问题，以及为什么这个问题需要解决；
    我觉得这部分还是非常重要的，解释了文章主要在解决一个什么样的研究问题，有时候也可以是自己的研究动机，从哪一个问题或角度出发考虑的。
  3. Others Methods: 第三段主要是介绍针对这个问题目前已有的方法，以及这些方法存在的问题，主要是为了与后面自己的方法形成对比，说明能够改善现有方法的劣势；
  4. Our Method: 第四段主要是介绍自己的方法，整体上怎样做的，然后为什么这个方法能够解决现有方法存在的问题；
  5. Contribution: 第五段主要是介绍自己方法的主要贡献，尤其是对这个研究领域的核心推进的地方，或者是一些对后续的研究启示。
